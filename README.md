# 🌈 `gradcat` 🐈

A [`lolcat`][lolcat] clone. 'Cuz why not?

## 📘 Usage

Give it a list of files to cat out or pipe it in through `stdin`.

```sh
gradcat ./src/main.rs
```

```sh
git log | gradcat
```
